# Changelog: Gemini Desktop Workspace

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-23

### Added
- **Unified Workspace Architecture**: Integrated `node-pty` to bridge the GUI file interactions with the Gemini CLI prompt engine.
- **High-Fidelity UI/UX**: Implemented a premium "Obsidian" grayscale aesthetic with `#0b0b0b` background and refined typography using Inter and JetBrains Mono.
- **Locked-Axis Sidebar**:
  - Established a strict vertical axis (X=30px) for core icons (Expand, New Chat, Nav items).
  - "Ghost" Expand Button: Dual-button logic where an anchored button remains stationary while a floating button transitions to the sidebar edge.
  - Zero-drift transitions during sidebar expansion/compression.
- **Precision Typography**: Strictly enforced 36px vertical rhythm (start-to-start) and 12-13px text height for sidebar items.
- **Custom Title Bar**: Discord-style custom header with centered "Gemini" branding, navigation controls, and transparent hover-highlight window buttons.
- **Smart Chat Input**:
  - `contenteditable` input field with inline image preview support.
  - Inline model selector triggered by a "person" icon.
  - Bidirectional file sync with `chokidar` (add files via Plus button or CLI).
- **Hardened Backend**:
  - Robust session management with automatic reload of the most recent session.
  - Fixed "White Screen" crashes by implementing data-type validation for `sessions.json`.
  - Floating popover system using `position: fixed` to escape container clipping.

### Fixed
- **Image Icon Bug**: Resolved issue where pasted images showed a "broken icon" in the chat input.
- **Sidebar Clipping**: Fixed user profile menu being cut off by sidebar overflow constraints.
- **Layout Overflow**: Applied strict viewport constraints (100vw/100vh) to prevent conversation text bar from being pushed off-screen.
