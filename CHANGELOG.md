# Changelog: Gemini Desktop Workspace

All notable changes to this project will be documented in this file.

## [1.2.0] - 2026-04-23

### Added
- **Primary Actions Group**: Consolidated "New chat", "Terminal", and "More" into a unified action cluster.
- **Dynamic Projects**:
  - Implemented project creation and management directly in the sidebar.
  - Added toggleable accordion logic with "fade up and out" transitions.
  - Created a high-fidelity "Folder Plus" icon with custom cutout aesthetics.
- **Next-Gen Typography**:
  - Migrated sidebar to **Instrument Sans** (ChatGPT aesthetic) at a precise **13px** scale.
  - Unified all sidebar text (headers, buttons, sessions) to a regular weight (400) for a cleaner, modern look.
- **Enhanced Interactivity**:
  - **Premium Hover**: Added rounded scale-down transitions to all sidebar elements while preserving label size.
  - **Contextual Terminal**: Integrated an "Up-Right" arrow indicator that appears on hover for the Terminal toggle.
  - **Unified Axis**: Realigned the entire sidebar content to a left-anchored `20px` padding scheme.

### Fixed
- **Transition Artifacts**: Eliminated millisecond text overlapping during sidebar collapse via instant visibility cut-offs.
- **Interaction Bugs**: Fixed phantom button clicks on the Gemini logo in expanded mode.

## [1.1.0] - 2026-04-23

### Added
- **Gemini Logo Integration**: Replaced the static expand button with a high-fidelity Gemini sparkle logo that anchors the sidebar.
- **Context-Aware Transitions**:
  - Implemented seamless hover transitions from the Gemini logo to the expand icon in compressed mode.
  - Added a "nearby radius" hover effect (100px hit area) to trigger the logo-to-icon transition as the cursor approaches.
- **Enhanced Sidebar Logic**:
  - **Dynamic Visibility**: The floating compress button now features a delayed fade-in when expanding and an early fade-out when compacting to prevent visual overlap with the anchored logo.
  - **Static Branding**: In expanded mode, the Gemini logo remains as a static brand element while the functional compress button appears independently on the right edge.

### Fixed
- **Sidebar State Desync**: Ensured the compress button is always visible when the app launches in expanded mode.

## [1.0.0] - 2026-04-23

### Added
- **Unified Workspace Architecture**: Integrated `node-pty` to bridge the GUI file interactions with the Gemini CLI prompt engine.
- **High-Fidelity UI/UX**: Implemented a premium "Obsidian" grayscale aesthetic with `#0b0b0b` background and refined typography using Inter and JetBrains Mono.
- **Locked-Axis Sidebar**:
  - Established a strict vertical axis (X=30px) for core icons (Expand, New Chat, Nav items).
  - "Ghost" Expand Button: Dual-button logic where an anchored button remains stationary while a floating button transitions to the sidebar edge.
  - Zero-drift transitions during sidebar expansion/compression.
- **Precision Typography**: Strictly enforced 36px vertical rhythm (start-to-start) and 12-13px text height for sidebar items.
- **Custom Title Bar**: Discord-style custom header with centered "Gemini" branding, navigation controls, and transparent hover-highlight window buttons.
- **Smart Chat Input**:
  - `contenteditable` input field with inline image preview support.
  - Inline model selector triggered by a "person" icon.
  - Bidirectional file sync with `chokidar` (add files via Plus button or CLI).
- **Hardened Backend**:
  - Robust session management with automatic reload of the most recent session.
  - Fixed "White Screen" crashes by implementing data-type validation for `sessions.json`.
  - Floating popover system using `position: fixed` to escape container clipping.

### Fixed
- **Image Icon Bug**: Resolved issue where pasted images showed a "broken icon" in the chat input.
- **Sidebar Clipping**: Fixed user profile menu being cut off by sidebar overflow constraints.
- **Layout Overflow**: Applied strict viewport constraints (100vw/100vh) to prevent conversation text bar from being pushed off-screen.
